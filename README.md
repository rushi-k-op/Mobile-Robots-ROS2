# Mobile Robots Portfolio

This repository tracks my coursework and personal projects focused on mobile robotics, autonomous navigation, and SLAM. All projects are built using ROS 2 and simulated in Gazebo (Otherwise specified).

### Global Tech Stack
* **Middleware:** ROS 2 (Humble)
* **Simulation:** Gazebo 3D Simulator
* **Visualization:** RViz2
* **OS:** Ubuntu 22.04

### How to use this repository
Each project lives in its own dedicated folder. If you want to run a specific simulation on your own machine, click into that project's folder and follow the instructions in its specific `README.md` file.
